# Angela Sprite Frames

This directory contains Angela's sprite frames for the Honor of Hero project.

## About

This directory was created to store Angela's in-game sequence frames (sprite animations). Currently, it contains a script to help download the sprites from a legitimate source.

## Download Script

The download_angela_sprites.py script is designed to download Angela's sprite frames from a legitimate source. To use it:

1. Install required dependencies:
   `ash
   pip install requests
   `

2. Edit the sprite_urls list in the script with actual URLs to Angela's sprite frames from a legitimate source

3. Run the script:
   `ash
   python download_angela_sprites.py
   `

## Important Notes

- Make sure you have permission to use any downloaded assets
- Only download assets from legitimate sources respecting copyright and terms of service
- The placeholder script is provided as an example framework
